# Substrate — Syntropic Design System

The design system for **Syntropic**, the world's first coherence soda. Built for one
job: shipping premium **websites and presentations** that look and feel like Syntropic —
deep violet substrate, cyan-violet emission glow, cathedral-quiet, anti-stimulant.

## Files

| File | What's in it |
|---|---|
| **DESIGN.md** | Core. Colors (OKLCH + HEX), surfaces, spacing, layout/grid, components, icons, do/don't, Tailwind v4 setup. Start here. |
| **TYPOGRAPHY.md** | Type. Geist + Geist Mono, full scale, rules, loading (web + presentations). |
| **MOTION.md** | Motion & interaction. Transition tokens, signature animations, UI/UX craft details, forbidden patterns, accessibility. |

## How to use

- **As a Claude skill:** push to GitHub, connect to your Claude Project. `DESIGN.md`
  carries the skill frontmatter and is the entry point; it references the other two.
- **As human reference:** open `DESIGN.md` and build. Each file is readable standalone.

## The idea in one line

The color is the mechanism: deep violet is the human-visible color of tryptophan UV
fluorescence in microtubules. Everything else follows from "light from within."
